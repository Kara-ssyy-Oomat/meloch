🔧 Восстановление порядка товаров

⏳ определяю режим...

📥 Шаг 1: Скачать порядок из локального кэша

Здесь в браузере уже лежат данные товаров с «правильным» порядком — нажми кнопку и они сохранятся в JSON-файл.

📤 Шаг 2: Загрузить порядок в Firestore

Выбери скачанный order-backup.json и нажми «Применить». Это работает прямо тут же — Firestore доступен по HTTPS даже с локального файла.

⚠️ Если Firebase откажется логинить с локального файла (ошибка auth/unauthorized-domain) — залей файл restore-order.html на GitHub и открой его там, тогда сработает.
